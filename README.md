- Requisitos
    - Python 3.10 o superior
    - Jupyter Notebook
    - pip


- Instalación
1. Recomiendo clonar el repositorio:
git clone (url-del-repo)

cd practica2-python

2. Crear un entorno virtual (Opcional):
python -m venv venv
Activar entorno:

    En Windows:
        venv\Scripts\activate
   
    En Linux/Mac:
        source venv/bin/activate

4. Instalar dependencias usando pip:
    pip install notebook ---> Instalara jupyter notebook


- Ejecución
1. Abrir Jupyter Notebook con el comando:

jupyter notebook (Si no se abre verificar que este correctamente instalado en el PATH)

3. Ir a la carpeta `notebooks/`

4. Ejecutar el archivo `ejercicio10.ipynb` para simular la competencia de cocina.

5. Cerrar jupyter notebook con Ctrl + C
